# Модерация аудио-приветствий

Административная панель для модерации голосовых приветствий в платных объявлениях.

## Расположение

Админ-панель → вкладка "Аудио" (иконка Volume2)

## Функциональность

### Статистика
- **На модерации** - количество аудио ожидающих проверки
- **Одобрено** - успешно прошедшие модерацию
- **Отклонено** - нарушающие правила платформы

### Три вкладки модерации

#### 1. На модерации
- Список объявлений с непроверенными аудио-приветствиями
- Кнопки "Одобрить" и "Отклонить" для каждого
- Бейдж с красным счетчиком на главной вкладке

#### 2. Одобрено
- История одобренных аудио
- Возможность отменить модерацию

#### 3. Отклонено
- История отклоненных аудио
- Показывается причина отклонения
- Возможность отменить модерацию

## Действия модератора

### Одобрение аудио
1. Прослушать аудио-приветствие
2. Убедиться что соответствует правилам
3. Нажать "Одобрить"
4. Аудио переходит в статус "Одобрено"

### Отклонение аудио
1. Прослушать аудио-приветствие
2. Выявить нарушение
3. Нажать "Отклонить"
4. В модальном окне указать причину отклонения
5. Подтвердить отклонение
6. Автор получит уведомление с причиной

### Отмена модерации
- Доступна для одобренных и отклоненных аудио
- Возвращает статус "На модерации"
- Используется при ошибочном решении

## Правила модерации

### ✅ Одобряется:
- Чистая речь без мата
- Соответствие описанию объявления
- Качественная запись (слышно слова)
- Соблюдение правил платформы

### ❌ Отклоняется:
- Нецензурная лексика
- Реклама сторонних ресурсов (сайты, телеграм-каналы)
- Спам или бессмысленные звуки
- Не соответствует описанию объявления
- Низкое качество (шум, неразборчиво)
- Нарушение правил платформы

## Информация в карточке объявления

- Аватар и имя автора
- Роль (мужчина/девушка)
- Дата создания объявления
- Цена и длительность
- Категория и тип (предложение/запрос)
- Бейджи (С голосом, В топе, статус модерации)
- Заголовок и описание
- Аудио-плеер для прослушивания
- Статистика (просмотры, отклики)
- ID объявления

## Уведомления

После модерации система отправляет уведомление автору:
- При одобрении: "Ваше аудио-приветствие одобрено"
- При отклонении: "Аудио отклонено: [причина]"

## Быстрые действия

**Keyboard shortcuts (будущая функция):**
- `A` - Одобрить выбранное
- `R` - Отклонить выбранное
- `U` - Отменить модерацию
- `Space` - Плей/Пауза аудио
- `↓/↑` - Следующее/предыдущее объявление

## Частые причины отклонения

Список для быстрого выбора в модальном окне:
1. Нецензурная лексика
2. Реклама сторонних ресурсов
3. Не соответствует описанию
4. Низкое качество записи
5. Нарушение правил платформы
